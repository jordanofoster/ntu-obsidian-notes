# Authentication

- Determining *who* someone is - done through various factors
	- Something you are
		- Biometrics
	- Something you have
		- Smartcard/Security Key, etc.
	- Something you know
		- Credentials - e.g., username/password
	- Some*where*/when you are
		- Location/Timezone checks
